# Contributing

We welcome contributions. Before your pull request can be merged, the following applies.

## Contributor Licence Agreement (CLA)

By opening a pull request, commenting with code, or otherwise contributing to this repository, you agree to the terms of our [Contributor Licence Agreement](./CLA.md).

In summary, You:
1. Grant Agentic Thinking Limited a broad licence to use your contribution.
2. Acknowledge that we may later relicence the work (for example, from Apache 2.0 to the Business Source License) subject to the **Apache Sunset Promise**: every Apache 2.0 release stays Apache 2.0 forever and the replacement licence converts back to Apache 2.0 on the fourth anniversary of each release.
3. Confirm you have the right to contribute (no employer or third-party conflicts).

No signature is required. Your submission of a pull request is deemed acceptance. We publish the CLA itself under Creative Commons CC0, so anyone may copy it for their own project.

## Our covenant

Alongside the legal CLA, we publish a [Community Covenant](./COVENANT.md) describing the cultural commitments we make to contributors. These include never requiring a signature, never introducing a corporate CLA, announcing licence changes 90 days in advance, and accepting community backports to the Apache line.

## How to contribute

1. Fork the repository.
2. Create a feature branch.
3. Make your change with clear, focused commits.
4. Open a pull request against `main`.
5. Include tests where applicable.

## Code of conduct

Be respectful. Technical criticism welcome, personal attacks are not.
